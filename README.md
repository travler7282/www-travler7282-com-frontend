# www-travler7282-com-frontend
This is the multi-framework front-end monorepo for travler7282.com, showcasing modern web development practices.

## Repository Structure
- `apps/landing-page`: The main entry point (Vanilla TS + Vite).
- `apps/react-app`: Vite + React implementation.
- `apps/vue-app`: Vite + Vue implementation.
- `apps/angular-app`: Angular CLI implementation.

## Quick Start
From the root directory:
`npm install` && `npm run build:all`
